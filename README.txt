# Телефонный справочник в консоли

Консольное приложение для управления справочником

После открытия в терминале будет предложены опции для дальнейшей работы

Выбирайте и следуйте инструкциям в консоли.

Для сохранения данных и выхода из программы выберите опцию "Сохранить и выйти"

Данные о записях в справочнике хранятся  в текстовом файле 'phonebook.txt'
